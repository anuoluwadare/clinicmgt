Using ClinicOps©
ClinicOps is a web-based application that Manages Clinic Operations and Medical information.
The users in ClinicOps are in 4 categories.

  1. Administrator (Admin)
  2. Doctor (Doctor)
  3. Nurse (Nurse)
  4. Pharmacist (Pharm)

The Admin role has access to the configuration and Administrative section of the application, the Doctor role has access to all the functions of a doctor while the Nurse role has access to all the nurse functions in the application. The Pharm role has access to pharmacist function. However note that roles can be combined to give a user multiple privileges. For example the admin and doctor roles can be combined to give a user access to the functions attached to those two roles.
This user manual will get you started on how to use the application.

Dispense Drugs
The Pharmacist or a user with Pharm rights is responsible for this section of the application. The dispensary has 3 modes namely:
Drug Request. This is when a patient comes directly to the dispensary to request for drugs, note that if the patient does not have a pending prescription from the doctor and there are no outstanding drug balances to collect. The application opens the drug request form. Also not that only drugs marked as on request will be allowed to be dispensed.
Drug Prescription. This is when a patient comes from the doctor with a prescription. The pharmacist can search for and dispense drugs to the patient on prescription and on request.
Partial Prescription. This is when a patient comes back to collect the balance of his/her drug if the patient has an outstanding drug to collect, this will be displayed to the pharmacist.

Searching for Drugs
The dispensary interface has a search engine for locating drugs. The drugs can be searched by drug groups by name and by a combination of multiple options. In drug prescription mode drugs returned are both on request and request however in Drug Request mode the drugs returned are only on request, drugs marked on prescription are not returned.
To Search: enter the search parameter and pick a criterion from the drop-down list and then click Search
